Real-Time Clinical Documentation Assistant

An AI-enhanced medical documentation application that leverages modern cloud-based speech recognition, natural language processing, and electronic health record (EHR) integration.

Real-time transcription of clinical notes using Deepgram’s ASR. These transcriptions are transformed into readable reports via ChatGPT and finally inserted as structured data into an EHR through Better EHR.

Example Workflow

  • User starts dictation via microphone.
  • Real-time transcription displays live, phrase-by-phrase.
  • After dictation, the transcript is reviewed and optionally edited.
  • Upon approval, the text is sent to CompliantChatGPT for formatting into a radiology report.
  • The final report can be modified and then inserted as structured data into the EHR.

The full process takes about a minute, primarily spent on dictation.

Technologies & Skills Demonstrated

  • Real-Time Transcription: Uses Deepgram ASR for low-latency performance without custom LLM fine-tuning.
  • Frontend Architecture: User interface using React for responsive transcription and interaction.
  • Structured Report Generation: Leverages HIPAA-compliant instances of ChatGPT to convert raw transcriptions into formatted clinical notes.
  • EHR Integration: Uses Better’s EHR system to demonstrate seamless FHIR-based data insertion.

Benefits

  • For Radiologists: Streamlines workflows and reduces transcription errors.
  • For Patients: Produces clear, accessible reports, enhancing understanding and satisfaction.
  • For Healthcare Admins: Cuts documentation costs and time, enabling better resource allocation.
