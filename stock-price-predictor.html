Stock Price Predictor

StockPricePredictor is a complete, GPU-accelerated machine learning workflow designed to build, train, and evaluate stock price prediction models using high-frequency 5‑minute OHLCV data.

This project showcases end-to-end capabilities across:

  • Data ingestion and preprocessing
  • Feature engineering and time-series transformation
  • Model training with GPU acceleration
  • Backtesting and strategy evaluation using vectorbt
  • Automated hyperparameter tuning with GPU‑accelerated grid search

Key Highlights

  • Ensemble Learning: Combines XGBoost and LSTM predictions for improved accuracy and robustness.
  • GPU Optimization: Utilizes CuPy, XGBoost (GPU backend), and TensorFlow with memory growth settings for efficient GPU use.
  • Command-Line Tooling: Launch training with a single command: 
    python -m src.models.train_model
  • Backtesting Integration: Employs vectorbt for signal-based strategy analysis using GPU outputs.
  • Modular Design: Lazily loads TensorFlow through a helper module to minimize unnecessary overhead.
  • Transparent Logging: Logs key training metadata, GPU usage, and ensemble diagnostics.

Technologies & Skills Demonstrated

  • Machine learning engineering with XGBoost and LSTM
  • GPU-accelerated pipelines using TensorFlow and CuPy
  • Time-series modeling for financial data
  • Backtesting with vectorbt
  • Automated training pipelines and CLI tools
