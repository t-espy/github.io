RAG Project

A full-stack Retrieval-Augmented Generation demo featuring Python + FastAPI for the API, React + Vite for the UI, Amazon Bedrock (Titan models) for embeddings and text generation, and FAISS for local vector search. The repository includes scripts to extract and chunk documents placed in datastore/ and to build the FAISS index using Bedrock embeddings. You can run the stack locally with Docker or deploy it to AWS via manual scripts or a CloudFormation template, with the deployed UI calling the Bedrock-backed API to answer questions.
