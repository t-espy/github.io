Continue Prompt Engineer

Continue Prompt Engineer is a Python-powered tool that analyzes Git repositories to automatically generate multi-shot prompt templates tailored for the Continue VS Code extension. By cloning and parsing Python codebases, it identifies recurring patterns, idioms, and coding styles to produce structured, high-quality prompt templates that preserve project-specific conventions. The tool supports intelligent pattern recognition, produces multiple usage examples per template, and outputs to a prompts/ directory for direct integration with Continue. Simple CLI usage and spaCy-based natural language processing make it easy to create consistent, context-rich templates from real-world code.

Key Features

  • Multi-shot Prompt Templates: Generates examples in Continue-compatible format
  • Code Pattern Recognition: Extracts idioms and coding styles from Python files
  • CLI-Based Workflow: Accepts a Git repo URL and optional output path
  • Style Preservation: Matches the project’s structure and conventions
  • Lightweight NLP: Uses spaCy (en_core_web_sm) for efficient code pattern detection

Technologies & Skills Demonstrated

  • Static code analysis with Python’s ast and tokenize modules
  • Natural language processing using spaCy with the en_core_web_sm model
  • Git integration using gitpython to clone and inspect repositories
  • Prompt engineering tailored for the Continue VS Code extension format
